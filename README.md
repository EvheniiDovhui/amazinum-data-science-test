Мої рішення тестових завдань для Amazinum. Тут три невеликі проєкти на Python.

Каталог Task 1 - пошук найглибшого місця в масиві висот. Знаходить заглиблення між піками та малює графік, зафарбовуючи «воду» блакитним.

Каталог Task 2 - є анімація (20 кроків), яка показує, як змінюється колонія клітин.

Каталог Task 3 - розрахунок ймовірності випадіння орла через правило Баєса. Програма «навчається» на попередніх кидках і видає прогноз для наступного.
